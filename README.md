# 📝 Exam Worksheet Generator

تطبيق ويب يولّد ملفات أوراق عمل مرتّبة من امتحانات PDF تلقائياً باستخدام Claude AI.

---

## 🎯 شو بيعمل؟

ترفع 3 ملفات:
1. **Question Paper (QP)** — ملف الامتحان الأصلي PDF
2. **Mark Scheme (MS)** — ملف الإجابات الصحيحة PDF
3. **شيت إكسل** — يحتوي على التصنيفات (التشابتر، مستوى الصعوبة، إلخ)

تضغط زر **Generate**، وبعد دقائق بتحمّل ملف Word (.docx) جاهز ومنسّق.

---

## 🚀 خطوات النشر (Deployment)

### الطريقة الأسهل: Streamlit Cloud (مجاني)

#### الخطوة 1️⃣: احصل على مفتاح Anthropic API

1. ادخل على https://console.anthropic.com
2. اعمل حساب جديد أو سجل دخول
3. اذهب إلى **Settings → Billing** وأضف رصيد ($5 كافي للبداية)
4. اذهب إلى **API Keys** واضغط **Create Key**
5. انسخ المفتاح (يبدأ بـ `sk-ant-...`) واحفظه في مكان آمن

> 💰 **التكلفة التقريبية**: كل امتحان من 40 سؤال ≈ $0.50 تقريباً (يعتمد على حجم الملفات)

#### الخطوة 2️⃣: ارفع الكود على GitHub

1. اعمل حساب على https://github.com (لو ما عندك)
2. اضغط على **New Repository** (الزر الأخضر)
3. اعطه اسم مثلاً: `exam-worksheet-generator`
4. خليه **Public** (عشان Streamlit Cloud المجاني يقدر يقرأ منه)
5. اضغط **Create Repository**
6. ارفع كل الملفات الموجودة في هذا المجلد على الريبو الجديد:
   - أسهل طريقة: اضغط **uploading an existing file** وارفع كل الملفات هنا (ما عدا `.streamlit/secrets.toml` لو موجود)

#### الخطوة 3️⃣: انشر على Streamlit Cloud

1. ادخل على https://share.streamlit.io
2. سجّل دخول بحساب GitHub
3. اضغط **New app**
4. اختر:
   - **Repository**: الريبو اللي عملته (مثلاً `your-username/exam-worksheet-generator`)
   - **Branch**: `main`
   - **Main file path**: `app.py`
5. اضغط على **Advanced settings** ➔ **Secrets**
6. الصق هذا في خانة الـ secrets:
   ```toml
   ANTHROPIC_API_KEY = "sk-ant-المفتاح-تبعك-هنا"
   ```
7. اضغط **Deploy**

⏰ بستغرق 2-3 دقايق للنشر. بعدها بطلعلك رابط مثل `https://your-app-name.streamlit.app` تقدر تستخدمه من أي مكان.

## 💻 التشغيل المحلي (لو بدك تجرب على جهازك)

تحتاج Python 3.9 أو أحدث.

```bash
# 1) ثبّت المكتبات
pip install -r requirements.txt

# 2) ضع مفتاح API في ملف .streamlit/secrets.toml
cp .streamlit/secrets.toml.example .streamlit/secrets.toml
# عدّل الملف وضع مفتاحك

# 3) شغّل التطبيق
streamlit run app.py
```

سيفتح التطبيق على http://localhost:8501

## 📋 شكل ملف الإكسل المطلوب

التطبيق يحاول يتعرف على الأعمدة تلقائياً. الأعمدة المتوقعة:

| اسم العمود (يمكن أحد هذه) | المحتوى |
|---|---|
| `Question No.` / `Question` / `Q` | رقم السؤال (1, 2, 3, ...) |
| `Topic` / `Chapter` / `Subject` | اسم التشابتر/الموضوع |
| `Difficulty` / `Level` | Easy / Medium / Hard |
| `Reference` / `Date` | تاريخ الامتحان (اختياري) |

## ❓ مشاكل شائعة

**"Invalid API key"** — تأكد من نسخ المفتاح كاملاً، ومن أن عندك رصيد في حساب Anthropic.

**"Couldn't parse Claude's response as JSON"** — أحياناً Claude بيرجع نص فيه شرح إضافي. اضغط Generate مرة ثانية.

**التطبيق بطيء** — معالجة كل امتحان بتاخد 2-5 دقائق حسب طول الامتحان. هذا طبيعي.

**الصور غير دقيقة** — الـ AI بيحاول يحدد مكان الصور تلقائياً، ودقّته متفاوتة. جرّب تحمل الملف وعدّل الصور يدوياً في Word.
